# Rectangle's area calculator ⏹️
It's a console app that calculates the area of a rectangle based on its height and length

# 💻 Programming Language: 

![C](https://img.shields.io/badge/c-%2300599C.svg?style=for-the-badge&logo=c&logoColor=white)

## 📦 Files

- `rectangle_area.c` → Source code of the program
- `rectangle_area.exe` → Executable file

## How to use ❓

1. Download the `.exe` file.
2. Run the program.
3. Enter the rectangle's length.
4. Enter the rectangle's height.
5. The program will display the area.
